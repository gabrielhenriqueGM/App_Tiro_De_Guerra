Escopo o Sistema
	O sistema deve propor o material usado nas intruções para
servir de fonte de consulta e/ou material de estudo prévio à
intrução, mantendo os AT's dos TG's conhecedor dos devidos assuntos.


Requisitos Funcionais
	RF01 - O sistema deve realizar uma conversão básico de Passos Duplos para Metro.
		RF01.1 - passoDuplo ≅ 100m ∴ passoDuploParaPercorrer = (distancia*passoDuplo)/100m.

			Regra de 3 simples (todos os valores em metros):
		             passoDuplo  ----  100m
		passoDuploParaPercorrer	 ----  distancia
